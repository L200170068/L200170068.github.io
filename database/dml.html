Data Manipulation Language

DML atau Data Manipulation Language adalah kumpulan perintah SQL yang berhubungan dengan pekerjaan mengolah data di dalam table.

    Berikut adalah perintah umum yang digunakan oleh DML:
  • Select adalah perintah yang paling sering digunakan pada SQL, sehingga kadang-kadang istilah query dirujukkan pada perintah select. Select digunakan untuk menampilkan data dari satu atau lebih tabel, biasanya dalam sebuah basis data yang sama.
  • Insert digunakan untuk menambah data baru.
  • Update digunakan untuk mengubah data yang sudah ada.
  • Delete digunakan untuk menghapus sebuah data.

Contoh SELECT:
SELECT * FROM peminjam_buku;

Contoh INSERT:
INSERT INTO peminjam_buku (nim, nama, tanggal_lahir, kota) values(L200170999, “Ardita″, “2013-01-11″, "Purwodadi");

Contoh UPDATE:
UPDATE peminjam_buku SET kota = “Surakarta” WHERE nim = L200170068;

Contoh DELETE:
DELETE FROM siswa WHERE nim = 'L2001700999';

Pada contoh perintah DML (Data Manipulation Language) di atas terdapat tiga perintah yang memiliki fungsi berbeda, pada baris pertama adalah perintah yang berfungsi untuk menapilkan data, di sana terdapat perintah utamanya adalah SELECT yang artinya menampilkan, kemudian di ikuti kode bintang yang artinya ‘semua’. Di ikuti kode FROM prminjam_buku yang memiliki arti dari Table peminjam_buku, Jika di baca secara lengkap akan berbunyi Menampilkan seluruh data yang terdapat pada table buku.

Kemudian ada baris kode yang berfungsi untuk menambahkan data, di sana terdapat fungsi INSERT INTO peminjam_buku, yang artinya data akan di masukan ke dalam table peminjam_buku yang di ikuti dengan nama-nama kolomnya secara urut. Kemudian di ikuti Values dengan isi data sesuai urutan nama kolomnya, terakhir di tutup dengan tanda petik dua pada setiap perintah SQL.

Pada baris ketiga terdapat perintah yang menggunakan kode utama UPDATE peminjam_buku yang artinya akan mengubah data yang sudah ada pada table peminjam_buku, di ikuti dengan SET kota = “Surakarta”, yang artinya data yang sudah ada akan di ubah dengan kata ‘Surakarta’. Lalu apa yang akan di ubah? yaitu terdapat pada kode terakhir WHERE nim=’L200170068′, kota yang akan di ubah dengan ‘Surakarta’ adalah kota yang memiliki nim L200170068.

Terakhir pada baris keempat digunakan untuk menghapus record dengan NIM ‘L200170999’, ingat bahwa NIM tersebut di gunakan oleh siswa bernama ‘Ardita’ dan dengan di eksekusinya perintah DELETE ini maka record tersebut akan terhapus.
